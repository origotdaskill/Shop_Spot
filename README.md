## About

This repository contains both client and server code. The client provides a seamless shopping interface, while the server handles backend operations.

## Features

- User-friendly interface
- Secure authentication
- Product management
- Shopping cart functionality
- Responsive design

## Installation

### Clone the repository

    ```bash
    git clone https://github.com/origotdaskill/Shop_Spot.git
    ```

## Navigate to the directories and install dependencies

### For the client:

    ```bash
    cd Shop_Spot/client
    npm install
    ```

### For the server:

    ```bash
    cd Shop_Spot/server
    npm install
    ```
## Usage
 ### Start the server
```bash
cd Shop_Spot/server
npm run dev 
```
 ### Start the client
```bash
cd Shop_Spot/client
npm start
```

Access the application at http://localhost:5173 .
